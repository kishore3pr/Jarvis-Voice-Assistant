# Jarvis Voice Assistant 🎙️

A Python-based voice assistant that can perform basic tasks using speech recognition.

## 🚀 Features
- 🎵 Play songs on YouTube
- ⏰ Tell current time
- 📚 Search Wikipedia
- 😂 Tell jokes

## 🛠️ Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- pywhatkit
- Wikipedia API

## ▶️ How to Run
```bash
pip install -r requirements.txt
python jarvis.py
